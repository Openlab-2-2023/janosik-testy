Zoznamy

Cieľom dnešného cvičenia je vytvoriť stránku, ktorá výzerá rovnakko ako táto.

Všetky potrebné informácie nájdete tu.

Vytvorenie nezoradeného zoznamu:

  • Vytvoríte nový HTML dokument s názvom napr.lists.html a inicializujete ho.
  • Nastavíte veľkosť písma na 20px
  • Vytvoríte header Zoznamy
  • Vytvoríte paragrafy s obsahom zhodným s príkladom
  • Vytvoríte header pre nezoradený zoznam
  • Vytvoríte zoznam pomocou <ul>
  • Vložíte obsah zhodný s príkladom pomocou tagu <li>

Vytvorenie zoradeného zoznamu

  1. Vytvoríte header pre zoradený zoznam
  2. Vytvoríte zoznam pomocou tagu <ol>
  3. Vložíte obsah zhodný s príkkladom pomocou tagu <li>
